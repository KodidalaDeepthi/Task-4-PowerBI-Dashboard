# Task-4-PowerBI-Dashboard
Interactive Dashboard 

📊 Sales Overview Dashboard | Power BI
📌 Project Overview

This project is an interactive Sales Overview Dashboard developed using Microsoft Power BI. The dashboard provides key business insights by analyzing sales performance, profit, quantity sold, customer segments, regions, categories, and yearly trends.

It helps stakeholders monitor business performance through interactive visualizations and dynamic filters.

🎯 Objective

The objective of this dashboard is to:

Analyze overall business performance.
Track Sales, Profit, and Quantity.
Compare sales across different regions and customer segments.
Identify top-performing product categories.
Monitor yearly sales and profit trends.
Enable users to filter data for better analysis.
🛠️ Tools Used
Microsoft Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Data Modeling

📂 Dashboard Features
KPI Cards
💰 Total Sales
📈 Total Profit
📦 Total Quantity Sold

Interactive Filters

Visualizations
